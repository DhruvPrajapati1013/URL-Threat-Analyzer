# URL Threat Analyzer

A Flask-based Cyber Security project that analyzes URLs and detects suspicious websites.

## Features

- URL Risk Analysis
- HTTPS Detection
- Domain & IP Detection
- Page Title Detection
- VirusTotal Integration
- Risk Score
- PDF Report Download

## Technologies

- Python
- Flask
- HTML
- CSS
- Requests
- BeautifulSoup
- ReportLab

## Installation

```bash
pip install -r requirements.txt
python app.py
```

Open:

```
http://127.0.0.1:5000
```
